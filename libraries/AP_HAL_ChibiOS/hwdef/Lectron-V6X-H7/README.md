# Lectron V6X Flight Controller

The Lectron is a flight controller produced by [Lectron](https://lectrontech.com/).

The [Lectron V6X](https://lectronuser.github.io/Lectron-Doc-Center/md/jetson/) is designed as an integrated flight control and computing platform for autonomous systems and advanced embedded applications. The hardware architecture consolidates real-time flight control and high-level computing into a single unified board.

This approach simplifies system integration, reduces cabling complexity, and improves overall system reliability.

## Features

- Processor
  - STM32H753 480Mhz, 
  - 2MB  flash memory
  - 32-bit processor
- Sensors
  - Dual ICM42670P IMU
  - Bosch BMI270 IMU
  - Dual Bosch BMP390 Barometers
  - Bosch BMM350 Magnetometer
- Power
  - **Input Voltage:** 12V – 25V (3S-6S LiPo)
